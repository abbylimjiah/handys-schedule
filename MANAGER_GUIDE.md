# 핸디즈 BQ 스케줄 사용 가이드 (매니저/HM용)

**URL**: https://abbylimjiah.github.io/handys-schedule/

---

## 🔐 1. 로그인

1. 사이트 접속 → 로그인 화면이 뜸
2. **회사 이메일** 입력 (예: `name@handys.co.kr`)
3. **닉네임** 입력 (예: Leo, Scarlett, Jenna 등 본인 영문명)
4. **입장하기** 클릭

> 💡 핸디즈 이메일(`@handys.co.kr`)이 없으면 접속 불가!

---

## 👥 2. 내 권한 확인

우측 상단 이름 옆에 배지 표시:

| 배지 | 의미 | 권한 |
|------|------|------|
| 🟡 **VIEWER** | 매니저 (일반) | 조회 + 메모 작성 |
| 🔵 **HM** (편집기간) | 헤드매니저 | 본인 지점 편집 |

---

## 📅 3. 편집기간 (HM만)

- **기본**: 매월 **20일~24일**에 다음달 스케줄 편집
- **2026년 5월 예외**: 4월 **17일~24일**
- 편집기간이 끝나면 자동으로 뷰어 전환

### HM 편집 권한
- 본인 소속 지점만 편집 가능 (다른 지점은 조회만)
- 헤더에 **"편집 가능 (HM)"** 초록 배지가 뜨면 편집 가능 상태

---

## 📝 4. 스케줄 입력 (HM 전용, 편집기간 중)

### 셀 클릭 → 근무유형 선택

**정규근무**
| 코드 | 시간 |
|------|------|
| D6 | 06:00~ |
| D9 | 09:00~ |
| M | 중간근무 |
| E | 14:00~ |
| N | 18:00~ |

**반차 / 반반차**
- `D6/반`, `D9/반`, `M/반`, `E/반`, `N/반` (반차 5종)
- `D6/반반`, `D9/반반`, `M/반반`, `E/반반`, `N/반반` (반반차 5종)

**휴무 / 연차**
- `#` 휴무
- `연차`, `대체`, `병가`, `공가`, `보건`, `경조`, `생일`, `출산`, `육아`, `태아`, `창립`

**기타**
- `파견`, `D9/단` (임산부 단축근무)

---

## 🗑 5. 스케줄 삭제 (HM, 편집기간 중)

### 다중 선택 삭제
1. 상단 **"벌크 모드"** ON
2. 삭제할 셀 여러 개 클릭 (파란 테두리 표시됨)
3. **🗑 삭제** 버튼 클릭

---

## 💬 6. 메모 작성 (누구나)

### 셀별 메모
- 셀 클릭 → 팝업에서 **메모** 입력
- 연차상신 / 카카오T 체크 가능 (마스터만)
- **뷰어도 메모 작성 가능!** ✨

### 지점 공통 메모
- 그리드 하단 **MEMO 행**에 작성
- 모든 직원이 볼 수 있는 공지용

---

## 🎨 7. 화면 보는 법

### 색상 의미
- 🟡 D6 (Amber)
- 🔵 D9 (Blue)
- 🟣 M (Violet)
- 🟠 E (Orange)
- 🟢 N (Teal)
- ⚫ # 휴무 (Gray)
- 🩷 연차 (Pink)
- 🔴 병가 (Red)

### 공휴일
- 빨간색 배경으로 하이라이트 표시
- 별도 "공휴일" 행에 이름 표시

### 오늘 날짜
- 파란색 하이라이트로 구분

---

## 📍 8. 지점 이동

### PC
좌측 사이드바에서 지점 클릭

### 모바일
상단 드롭다운에서 지점 선택

> 💡 HM은 다른 지점 볼 순 있지만 편집은 본인 지점만!

---

## ❓ 9. 자주 묻는 질문

**Q. 편집이 안 돼요!**
→ 편집기간(20~24일)인지 확인. HM만 편집 가능.

**Q. 다른 지점도 편집하고 싶어요**
→ 운영지원팀(Abby)에게 추가 권한 요청

**Q. 스케줄이 안 보여요**
→ 월 선택기에서 원하는 월 선택했는지 확인

**Q. 휴대폰에서도 사용 가능해요?**
→ ✅ 가능! PC/태블릿/모바일 모두 지원

**Q. 로그아웃하고 싶어요**
→ 우측 상단 "로그아웃" 버튼 (마스터만 표시)

**Q. 데이터는 어디 저장되나요?**
→ 현재는 본인 브라우저 (localStorage)
→ 추후 회사 서버 연동 예정
