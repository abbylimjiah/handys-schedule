# 핸디즈 BQ 스케줄 관리 시스템 사용설명서

**URL**: https://abbylimjiah.github.io/handys-schedule/

---

## 1. 로그인

### 일반 사용자
- 회사 이메일 (`@handys.co.kr`)로 로그인
- 닉네임은 본인 영문명 (예: Abby, Leo)

### 마스터 (운영지원팀)
- 우측 상단 **"마스터"** 버튼 클릭
- 마스터 비밀번호 입력 (초기: `handys2026`)

---

## 2. 사용자 역할

| 역할 | 권한 |
|------|------|
| **MASTER** | 전체 편집, 사용자 관리, 비밀번호 변경, 아마란스 다운로드 |
| **BRANCH** | 지정된 지점만 편집 가능 (편집기간 중) |
| **VIEWER** | 조회 + 메모 작성만 가능 |
| **HM 자동** | 편집기간 중 본인 소속 지점 자동 편집 권한 |

---

## 3. 편집기간

- **기본**: 매월 **20일~24일** (다음달 스케줄 편집)
- **2026년 5월 예외**: 4월 17일~24일
- 편집기간이 끝나면 자동으로 뷰어 모드 전환
- 마스터는 언제든 **"편집 기간 오버라이드"** 체크로 수동 활성화 가능

---

## 4. 스케줄 입력

### 셀 클릭 → 근무유형 선택
- **정규근무**: D6, D9, M, E, N
- **반차**: D6/반, D9/반 등 (5종)
- **반반차**: D6/반반, D9/반반 등 (5종)
- **휴무/연차**: #, 연차, 대체, 병가, 공가, 보건, 경조, 생일, 출산, 육아, 태아, 창립
- **기타**: 파견, D9/단(임산부)

### 다중 선택 삭제
1. 좌측 상단 **벌크 모드** ON
2. 여러 셀 선택 후 **🗑 삭제** 버튼

---

## 5. 인원 관리 (마스터 전용)

### 직원 추가/수정/삭제
- 헤더 **"인원 관리"** 버튼
- 이름 더블클릭으로 인라인 수정
- 입사일 / 직책(Lead/HM/Mgr) 편집

### 권한 관리
- 헤더 **"권한 관리"** 버튼
- 총 155명 전체 사용자 목록 표시
- **편집**: 역할/이메일/지점 변경
- **차단**: 로그인 막기
- **+ 사용자 추가**: 직원 외 인원(인사팀 등) 추가

---

## 6. 아마란스 엑셀 다운로드 (마스터 전용)

- 헤더 **📥 아마란스** 버튼
- **현재 지점**: 선택된 지점만 다운로드
- **전체 지점**: 30개 지점 통합 다운로드
- CSV 형식 (Excel에서 열림)
- 근무유형 자동 변환: D9→001, E→002, #→004, M→005, D6→006, N→007

---

## 7. 공휴일
2026년 19개 공휴일 자동 표시 (그리드에 빨간색 하이라이트)

---

## 8. 메모 & 연차상신

### 셀 클릭 시 팝업
- **연차상신**: 체크 시 해당 날짜 연차 신청 표시
- **카카오T**: 출퇴근 인증 체크
- **메모**: 자유 입력 (뷰어도 작성 가능)

### MEMO 행
- 그리드 하단에 지점 공통 메모 작성 가능

## 9. 자주 묻는 질문

**Q. HM인데 편집이 안 돼요?**
→ 편집기간(20~24일)에만 편집 가능. 5월은 4/17~24일.

**Q. 비밀번호를 잊어버렸어요?**
→ 초기 비번 `handys2026`. 변경했다면 localStorage 초기화 필요.

**Q. 데이터는 어디 저장되나요?**
→ 현재는 브라우저 localStorage (개인 기기에만). 추후 서버 DB 연동 예정.

**Q. 사용자를 차단하면?**
→ 차단된 이메일로 로그인 시도 시 "차단된 계정" 메시지 표시.
